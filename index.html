Документация API

Метод `get_game_list.php`

Этот метод возвращает список игр с информацией о каждой игре.

Запрос

  • URL: http://rusty32307.tw1.ru/api/get_game_list.php
  • Метод: GET

Ответ

JSON-массив объектов, каждый объект представляет игру:


[
  {"GameID":"1","ImageSource":"...","Title":"MyRust 220","Status":"Null"},
  {"GameID":"2","ImageSource":"...","Title":"YRS Rust 136","Status":"Античит: YAC"},
  ...
]
